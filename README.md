# just4media

[Demo](https://just4media.herokuapp.com/)

### Run This Project In Local Machine: 
 
 

1.  Clone the repo:

    ## SSH:
        ```
        git@github.com:walaamedhat/just4media.git
        ```
        
    
    ## HTTPS:
        ```
        https://github.com/walaamedhat/just4media.git
        ```
2.  Install the node.js devDependencies:
    ```
    yarn install
    ```
    OR 
    ```
    yarn
    ```
3.  Run the Server:
    ```
    npm start
    ```
    OR 
    ```
    npm run start
    ```
